# Panduan Menjalankan Project Tanpa Docker (Local XAMPP/Laragon)

Panduan ini ditujukan untuk menjalankan aplikasi **Inventori MSU** di lingkungan lokal menggunakan XAMPP atau Laragon, tanpa menggunakan Docker.

## 1. Prerequisites (Persyaratan)

Pastikan software berikut sudah terinstall di komputer Anda:

1.  **XAMPP** atau **Laragon** (untuk PHP & MySQL).
    *   PHP versi **8.1** atau lebih baru.
    *   MySQL / MariaDB.
2.  **Composer** (Manajer dependensi PHP).
    *   Download: [getcomposer.org](https://getcomposer.org/)
3.  **Node.js & NPM** (Untuk compile aset frontend).
    *   Download: [nodejs.org](https://nodejs.org/) (Versi LTS disarankan).
4.  **Git** (Untuk clone/pull repository).

---

## 2. Persiapan Database

1.  Jalankan modul **Apache** dan **MySQL** di XAMPP/Laragon.
2.  Buka **phpMyAdmin** (biasanya di `http://localhost/phpmyadmin`).
3.  Buat database baru dengan nama: `inventori_msu`
    *   Collation: `utf8mb4_unicode_ci` (opsional tapi disarankan).

---

## 3. Instalasi Project

Buka terminal (Command Prompt / PowerShell / Git Bash) dan arahkan ke folder project `inventori-msu`.

### Langkah 1: Setup Environment (.env)

Saya telah menyediakan file konfigurasi khusus lokal bernama `.env.local`. Salin file ini menjadi `.env`:

*   **Windows (PowerShell):**
    ```powershell
    copy .env.local .env
    ```
*   **Manual:**
    Duplicate file `.env.local`, lalu rename hasil copy-nya menjadi `.env`.

> **Catatan:** Pastikan setting database di file `.env` sudah sesuai dengan XAMPP Anda (default user `root` tanpa password biasanya sudah cocok):
> ```env
> DB_CONNECTION=mysql
> DB_HOST=127.0.0.1
> DB_PORT=3306
> DB_DATABASE=inventori_msu
> DB_USERNAME=root
> DB_PASSWORD=
> ```

### Langkah 2: Install Dependensi PHP (Composer)

Install library Laravel yang dibutuhkan:

```bash
composer install
```

### Langkah 3: Generate Application Key

```bash
php artisan key:generate
```

### Langkah 4: Setup Database (Migrate & Seed)

Jalankan perintah ini untuk membuat tabel dan mengisi data awal (user admin, barang dummy, dll):

```bash
php artisan migrate:fresh --seed
```

### Langkah 5: Setup Storage Link

Agar gambar yang diupload bisa diakses publik:

```bash
php artisan storage:link
```

### Langkah 6: Install Dependensi Frontend (NPM)

Install library Javascript/CSS:

```bash
npm install
```

---

## 4. Menjalankan Aplikasi

Anda perlu menjalankan **dua terminal** secara bersamaan.

**Terminal 1 (Laravel Server):**
Menjalankan server backend PHP.
```bash
php artisan serve
```
*App akan berjalan di: [http://localhost:8000](http://localhost:8000)*

**Terminal 2 (Vite Development Server):**
Menjalankan hot-reload untuk CSS/JS (agar tampilan update otomatis saat diedit).
```bash
npm run dev
```

---

## 5. Akses Aplikasi

Buka browser dan kunjungi: **[http://localhost:8000](http://localhost:8000)**

*   **Login Admin:** [http://localhost:8000/login](http://localhost:8000/login)
*   **Halaman Booking (Guest):** [http://localhost:8000/booking-barang](http://localhost:8000/booking-barang)

---

## Troubleshooting Umum

*   **Error "Access denied for user 'root'@'localhost'":**
    Cek file `.env`, pastikan `DB_PASSWORD` kosong (jika XAMPP default) atau sesuaikan jika Anda pernah mengubah password root MySQL.
*   **Gambar tidak muncul:**
    Pastikan sudah menjalankan `php artisan storage:link`.
*   **Tampilan berantakan / CSS hilang:**
    Pastikan `npm run dev` sedang berjalan di terminal terpisah.
